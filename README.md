# サンプルリポジトリ

このリポジトリは、GitHubの基本的な機能を学習するためのサンプルプロジェクトです。

## 機能

- GitHubの基本操作の練習
- Markdownの記法の確認
- プロジェクト管理の基礎学習

## 使い方

1. このリポジトリをクローンします
```bash
git clone https://github.com/krpsV/sample-repository.git
```

2. 必要な変更を加えます

3. 変更をコミットしてプッシュします
```bash
git add .
git commit -m "変更内容の説明"
git push origin main
```
